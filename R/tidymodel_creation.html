Create Model using tidymodels

For this example, we are simply going to create a tidymodel using logistic regression. If youd like to learn more about training/testing/evaluating a model, please visit the tidymodels website.

Define model specification and build workflow

# Use logistic regression (glm)
glm_spec <- logistic_reg() %>% 
  set_engine("glm")

# Build Workflow
penguin_wf <- workflow() %>% 
  add_formula(sex ~ .)

Create Model

# Split data into training and testing
set.seed(123)
penguin_split <- initial_split(penguins_df, strata = sex)
penguin_train <- training(penguin_split)
penguin_test <- testing(penguin_split)

# Create model on trainig set
penguins_model <- penguin_wf %>% 
  add_model(glm_spec) %>% 
  fit(penguin_train)

# Validate on testing set
pred_val <- penguins_model %>% 
  predict(penguin_test) %>% 
  bind_cols(penguin_test)

conf_mat(pred_val, sex, .pred_class)
